# Omnichannel Messaging MVP

Phase 1 MVP for omnichannel messaging platform using hexagonal architecture.

## 🎯 Project Overview

Currently supports WhatsApp messaging via Gupshup and Meta providers.

## 🏗️ Architecture

- **Hexagonal Architecture** with explicit ports and adapters
- **Domain-Driven Design** with rich domain models
- **Microservices** with clear boundaries
- **Event-driven** communication

## 📂 Structure

```
omnichannel-messaging-mvp/
├── apps/                   # Microservices
├── libs/                   # Shared libraries
└── infrastructure/         # DevOps & deployment
```
